# Premium Trainer Site

Ultra-modern premium website for a personal fitness trainer.

## Stack
- Next.js (App Router)
- React + TypeScript
- Tailwind CSS
- Framer Motion

## Run
1. Install Node.js 20+
2. In this folder run:
   - npm install
   - npm run dev
3. Open http://localhost:3000

## Build
- npm run build
- npm start
